# CryptoPulse Canvas Analytics

Интерактивный SPA-дашборд для отслеживания стоимости популярных криптовалют (Bitcoin, Ethereum, Solana) за различные периоды времени (1, 7, 30, 90 дней).
Приложение спроектировано с использованием методологии **Feature-Driven Design (FSD)** на базе фреймворка **Next.js 15+** (React 19). Интерактивный график цен отрисован на чистом **HTML5 Canvas API** без использования тяжеловесных сторонних графических библиотек.

## Архитектура проекта (Feature-Driven Design)

Кодовая база строго разделена на слои и слайсы в соответствии с методологией FSD, что обеспечивает высокую масштабируемость, слабую связанность модулей и контролируемое направление импортов:

- **`app/`** - Системный слой Next.js (App Router). Содержит конфигурацию роутинга страниц, корневой лэйаут и глобальные стили (`globals.css`).
- **`src/features/`** - Слой фич (интерактивные действия пользователя):
  - `select-chart-period/` - Переключение таймфреймов графиков (1, 7, 30, 90 дней).
  - `select-coin/` - Смена активной криптовалюты (Bitcoin, Ethereum, Solana).
- **`src/entities/`** - Слой бизнес-сущностей:
  - `crypto-chart/` - Центральный слайс графиков. Включает Canvas-компонент, карточки финансовой статистики (`StatCards`), кастомный хук получения данных и TypeScript типы.
- **`src/shared/`** - Слой переиспользуемого инфраструктурного кода:
  - `ui/ThemeToggle` - Модуль переключения темной темы (Dark Mode).

_Каждый слайс предоставляет строго изолированный интерфейс через файлы **Public API (`index.ts`)**._

## Технологический стек

- **Framework:** Next.js 15+ (React 19, Server & Client Components)
- **Styling:** Tailwind CSS v4 (с поддержкой семантических токенов и плавной темной темы)
- **Graphics:** HTML5 Canvas API (кастомная математика динамического масштабирования)
- **Data Source:** CoinGecko API (с обработкой Демо-ключей и лимитов)
- **Testing:** Vitest & React Testing Library (юнит-тесты логики данных)

## Ключевые фичи

1.  **Отказоустойчивость (Fault Tolerance):** Внутри кастомного хука `useCryptoChartData` реализована система автоматического переключения на локальный математический генератор данных (Mock), если внешний сервер CoinGecko API недоступен или превышен лимит запросов (ошибка 429/500).
2.  **Интерактивный Zoom & Pan:** Кастомная реализация масштабирования (колесиком мыши) и панорамирования (перетаскиванием зажатой левой кнопкой мыши) внутри Canvas API с фокусом на курсор (по принципу TradingView) и автоматическим динамическим сдвигом дат по оси X.
3.  **Детерминированный Random (LCG):** Для генерации резервных данных используется линейный конгруэнтный генератор псевдослучайных чисел (Linear Congruential Generator) с фиксированным сидом (`seed`). Это исключает случайный хаос и делает генерацию тренда абсолютно идентичной при каждом перезапуске, что критически важно для стабильности тестов.
4.  **Оптимизация Canvas в Dark Mode:** Компонент холста слушает кастомное событие `themechange` и налету перерисовывает внутренние пиксели (сетку осей, шрифты шкал, градиенты подложки) в соответствии с активной темой, не вызывая утечек памяти.
5.  **Интерактивный Tooltip:** Математический расчет ближайшей точки (X/Y) относительно курсора мыши для вывода точной цены "на лету" без просадки FPS и без использования DOM-элементов под каждую точку.
6.  **Адаптивные шкалы:** Динамическое переключение сетки (отображение 5-минутных интервалов времени для 1 дня и календарных дат для больших периодов).
7.  **Retina-адаптивность:** Разрешение Canvas динамически пересчитывается через `ResizeObserver` и масштабируется с учетом физической плотности пикселей экрана устройства (`window.devicePixelRatio`). График остается четким, как на 4K-мониторах, так и на экранах смартфонов.

## Покрытие тестами (Vitest & RTL)

Приложение покрыто автоматическими тестами, разделенными на два уровня:

- **Unit-тесты:** Проверка математической логики и структуры детерминированного генератора цен (`generateMockData.test.ts`).
- **Integration-тесты:** Эмуляция среды браузера (`jsdom`) для проверки полной сборки страницы `page.tsx` (`page.test.tsx`). Тесты проверяют асинхронное состояние загрузки (`Loading...`), корректность математических расчетов в карточках статистики на основе фикстур, а также успешный откат в резервный режим при сбоях сети.

Запуск тестов:

```bash
npm run test
```

## Как запустить локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ochuksin/coin-pulse
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл `.env.local` в корне проекта и добавьте ваш демо-ключ CoinGecko:
   ```env
   NEXT_PUBLIC_COINGECKO_API_KEY=CG-xxxxxx
   ```
4. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
