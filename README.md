# 📈 CryptoPulse Canvas Analytics

Интерактивный SPA-терминал для отслеживания стоимости криптовалют, построенный на Next.js (App Router) и Tailwind v4 с отрисовкой графиков на чистом HTML5 Canvas без использования сторонних библиотек.

## Технологический стек

- **Framework:** Next.js 15+ (React 19, Server & Client Components)
- **Styling:** Tailwind CSS v4 (с поддержкой семантических токенов и плавной темной темы)
- **Graphics:** HTML5 Canvas API (кастомная математика динамического масштабирования)
- **Data Source:** CoinGecko API (с обработкой Демо-ключей и лимитов)
- **Testing:** Vitest & React Testing Library (юнит-тесты логики данных)

## Ключевые фичи

- **Отказоустойчивость (Fault Tolerance):** Автоматическое переключение на математический генератор случайных блужданий (Mock Data), если внешний API недоступен или превышен лимит запросов.
- **Оптимизация кэширования:** Использование Next.js Request Deduplication для параллельных серверных вычислений.
- **Интерактивный Tooltip:** Математический расчет ближайшей точки (X/Y) относительно курсора мыши для вывода точной цены "на лету" без просадки FPS.
- **Адаптивные шкалы:** Динамическое переключение сетки (отображение 5-минутных интервалов времени для 1 дня и календарных дат для больших периодов).
